    最近在迁移博客到 hexo，执行 hexo server 会有警告，而且 hexo generate 无法生成页面。后来发现是于node 14不兼容的原因，考虑安装低版本node，比如node 12.16.3 LTS版本。

    macOS环境下，使用homebrew安装的node，默认是最高版本，命令如下：

    那么如何使用homebrew安装指定版本的node呢?

    如果之前使用 brew install node 安装过node,需要先执行 brew unlink node 来’解绑’node
    查找可用的node版本 brew search node
    安装你需要的版本, 比如 brew install node@12
    然后 brew link node@12 ，这一步可能会报错，按照提示执行命令就ok了，比如我最后执行的是 brew link --overwrite --force node@12
    node -v 不出意外, 就安装好了你想要的node版本
